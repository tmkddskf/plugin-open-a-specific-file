# Open a Specific File

一个简单的 Obsidian 插件，让你可以快速打开指定的文件。

## 功能

- 📄 快速打开指定文件：通过命令或侧边栏图标快速打开你配置的文件
- 📅 每日笔记：快速打开今天的每日笔记（需要启用 Daily Notes 插件）

## 安装

1. 打开 Obsidian 设置
2. 进入社区插件
3. 关闭安全模式（如果开启）
4. 点击浏览并搜索 "Open a Specific File"
5. 点击安装
6. 在社区插件列表中启用该插件

## 使用方法

### 配置快速打开文件

1. 打开设置
2. 进入 "Open a Specific File" 设置页面
3. 点击 "Add File Configuration" 添加文件配置
4. 填写：
   - 名称：显示名称
   - 路径：文件在 vault 中的相对路径

### 打开文件的方式

- 使用命令面板（Ctrl/Cmd + P）搜索 "Open [文件名]"
- 点击左侧栏的文件图标打开文件列表

## 系统要求

- Obsidian v0.15.0 或更高版本
- Daily Notes 插件（可选，用于每日笔记功能）
